# KA Pointer

スクリーンポインタハイライトアプリ。ScreenPointer の代替。

## インストール

```bash
git clone https://github.com/ka-1311/claude.git
cd claude
bash build-pointer.sh
open "/Applications/KA Pointer.app"
```

初回起動後:
1. **システム設定 > プライバシーとセキュリティ > アクセシビリティ** で許可
2. **システム設定 > 一般 > ログイン項目** で「KA Pointer」を追加（Mac起動時に自動起動）

## 操作

| 操作 | トリガー |
|------|---------|
| スポットライト ON/OFF | 左Cmd+右Cmd（トグル） |
| 矩形ハイライト ON/OFF | 左Cmd+右Cmd+Control（トグル） |
